# vue-template

vue1.0.26 感谢noop

模板，包含webpack,jquery,bootstrap,vue-router,vue-resource,vuex

# webpack-dev-server

使用webpack的服务，所以全局安装

```
npm install webpack-dev-server -g
```

<http://webpack.github.io/docs/webpack-dev-server.html>

```
    npm install
```

# 运行

```
    npm run dev
```

# 打包

```
    npm run pack
```

打包需要使用webpack命令，所以请全局安装

```
    webpack npm install webpack -g
```
